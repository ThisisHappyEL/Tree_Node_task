# Tree_Node_task
Для проверки работоспособности бинарного дерева в файле `main.py` создан экземпляр для проверки.

Чтобы убедиться в соответствии результата ожиданиям, вам потребуется:

1. Скопировать репозиторий себе.
2. Установить зависимости проекта командой `uv sync`.
3. Запустить код в файле `main.py` командой `uv run main.py` (или `python3 main.py`), находясь в директории проекта.
